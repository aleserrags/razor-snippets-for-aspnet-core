# Razor Snippets

Razor Snippets for Asp.Net Core

## Usage

Type part of a snippet, press `enter`, and the snippet unfolds.

### Snippets

| Conditional Expressions      | 
| ---------------------------- |
| `@if`                        |
| `@ifelse`                    |
| `@switch`                    |

| Repetition Structures        | 
| ---------------------------- |
| `@for`                       |
| `@foreach`                   | 
| `@while`                     | 
| `@do`                        |

| HTML Helpers                 | 
| ---------------------------- |
| `@Html.DisplayNameFor`       |
| `@Html.DisplayFor`           |
| `@Html.ActionLink`           | 
| `@Html.Action`               | 
| `@Html.AntiForgeryToken`     | 
| `@Html.CheckBox`             |
| `@Html.CheckBoxFor`          |

| Others                       | 
| ---------------------------- |
| `@ViewBag`                   | 
| `@ViewData`                  | 
| `@RenderBody`                | 
| `@model`                     |
| `@RenderSection`             |
| `@section`                   |
